# JP web novels reader (WIP)
Веб-приложение предоставляет удобный способ поддерживать библиотеку веб-новел, опубликованных на различных японских веб-сайтах, таких как Narou (なろう), Kakuyomu (カクヨム) и т.п. Вы сможете комфортно читать новели и вести учет прочитанных глав.

Основные возможности:
* Управление библиотекой
* Чтение веб-новел
* Отслеживание прогресса
* Поддержка как вертикального, так и горизонтального направления письма
* Программно реализован горизонтальный скролл колесиком мышки (при чтении в режиме вертикального письма)
* Настройки ридера (Размер шрифта, темы, направление письма, line-height, etc)
